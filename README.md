# 📱 PhonePe Transaction Analysis Dashboard

## 📌 Overview

This project focuses on analyzing PhonePe transaction data using Microsoft Excel, Power Query, and Power BI. 
The objective was to transform raw transaction data into meaningful business insights through data cleaning, modeling, and interactive dashboard creation.

The dashboard provides a comprehensive view of transaction performance across multiple PhonePe services, 
helping stakeholders monitor transaction trends, payment success rates, failure reasons, and service-wise contributions.

## 🎯 Business Objective

The primary objectives of this project are:

- Analyze transaction performance across different PhonePe services.
- Monitor successful and failed transactions.
- Identify major reasons for payment failures.
- Track monthly transaction trends.
- Compare service-wise transaction amounts and volumes.
- Build an interactive dashboard for data-driven decision-making.

# 📊 Dashboard Analysis

## 1️⃣ Transaction Overview Dashboard

### KPIs
- Total Amount: 3,333M
- Total Transactions: 300K
- Successful Transactions: 288K
- Failed Transactions: 12K

### Insights
- Loans generated the highest transaction amount.
- July recorded the highest monthly transaction amount.
- More than 95% of transactions were successful.
- Server Errors, Wrong PIN, and Insufficient Balance were the leading causes of failed payments.

---

## 2️⃣ Insurance Dashboard

### Categories Analyzed
- Bike Insurance
- Car Insurance
- Term Life Insurance
- Health Insurance

### Insights
- Insurance transactions remained consistent throughout the year.

- October recorded the highest insurance transaction value.

## 3️⃣ Loan Dashboard

### Categories Analyzed

## 4️⃣ Money Transfer Dashboard

## 5️⃣ Recharge & Bills Dashboard

### Categories Analyzed
- Electricity
- Mobile Recharge
- DTH
- Cable TV

### Insights
- Electricity and DTH generated the highest revenue contribution.

---

# 📈 Key Findings

- Over 95% of transactions were completed successfully across all services.
- Loan services contributed the highest transaction value.
- UPI Transfers were the most frequently used money transfer method.
- Electricity and DTH dominated the Recharge & Bills category.
- Wrong PIN, Server Error, and Insufficient Balance were the major payment failure reasons.
- Transaction amounts showed seasonal peaks during mid-year months.
